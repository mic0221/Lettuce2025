# Lettuce2025
This repository contains Python scripts used for image preprocessing and segmentation models for lettuce tipburn quantification as well as a R markdown file for data visualization in the MSc thesis project: Quantification and genetic mapping of tipburn in Lactuca serriola (2025).

## Environment
The project uses a Conda environment. The environment for segmentation model usage and GWAS can be found in environment_model.yml and environment_GWAS.yml respectively.
